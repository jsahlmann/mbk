## SAS program snippet

The following SAS code will be executed.
proc mixed data=df07 method=reml covtest;
   class Group (Ref = "Group_1") Time (Ref = "Time_1") USUBJID;
   model Y_comp = Group Time Group*Time / s;
   repeated / type=un subject=USUBJID r;
run;
